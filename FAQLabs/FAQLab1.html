Frequently asked questions

I cannot login and my password is right?

Check your syntax, usually you might be entering ssh RCSID@FQDN in my case it will be ssh kcs@kcs.easturpi.com NOT ssh kcs.eastus.rpi.com although, sometimes the ssh without rcsid works as well. most of the time it is this problem.

I'm getting the error "access denied"

Use sudo command(which means superuser do) before your command, for example: ls becomes sudo ls

Can I use my existing GitHub account?

Yes!, make sure to add rpi as a secondary email
